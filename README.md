# ShopEase – Modern E-Commerce Store

A beautiful, fast, and fully functional e-commerce platform with product listing, cart system, user authentication, and admin features.

### Tech Stack & Framework
- **Framework**: Next.js 
- **Styling**: Tailwind CSS + Framer Motion (animations)  
- **Backend**: Node.js + Express + MongoDB (Mongoose)  
- **Authentication**: NextAuth.js (Google + Credentials)  
- **State Management**: React Context (Cart)  
- **Deployment**: Frontend → Netlify | Backend → Vercel  
- **Icons & UI**: Lucide React + React Hot Toast
